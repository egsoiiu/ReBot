## ğŸ“– Usage

### How to Rename Files

1. **Send a file** to the bot (document, video, or audio)
2. **Click "Rename"** button
3. **Enter new filename** (without extension)
4. **Select upload type** (Document or Video)
5. **Wait for processing** with progress tracking

### Thumbnail Management

- **Set thumbnail**: Simply send a photo to the bot
- **View thumbnail**: Use `/view_thumb` command
- **Delete thumbnail**: Use `/del_thumb` command
